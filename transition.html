Переходы

Изменение стиля элемента в момент изменении его состояния

transition-duration

Определяет промежуток времени, в течение которого должен осуществляться переход. В значении указывается время за которое одно значение свойства переходит в другое (s, ms). Свойство не наследуется.


transition-property

Содержит название CSS-свойств, к которым будет применен эффект перехода. Значение свойства может содержать как одно свойство, так и список свойств через запятую. Свойство не наследуется.
При значении all будут анимироваться все свойства

transition-delay

Задержка выполнения перехода. Позволяет сделать так, чтобы изменение свойства происходило не моментально, а с некоторой задержкой. Время задержки перехода указывается в секундах или миллисекундах. Свойство не наследуется.


transition-timing-function

Задаёт стиль нашего перехода, то есть сценарий, по которому будет осуществяться анимация.

Некоторые значения:

  • ease - переход начинается медленно, разгоняется быстро и замедляется в конце.
  • ease-in - переход начинается медленно, а затем плавно ускоряется в конце.
  • ease-out - переход начинается быстро и плавно замедляется в конце.
  • ease-in-out - переход медленно начинается и медленно заканчивается.
  • linear - переход происходит равномерно на протяжении всего времени, без колебаний в скорости.

transition

Для элемента можно задать несколько последовательных переходов, перечислив их через запятую. Каждый переход можно оформить своими значениями перехода.
Синтаксис:
transition: property duration timing-function delay


Анимация

Использование конструкции animation и @keyframes позволяет создавать более сложные сценарии анимаций (автоматическое воспроизведение, повторы, остановка по определенному событию).

animation-name

Определяет список применяемых к элементу анимаций (ключевых кадров). Можно указать через запятую. Причем приоритет у последней записи. Если нужно отменить анимацию, ставим значение none. animation-name: none; 

                     .animation-circle{
                        width: 100px;
                        height: 100px;
                        border-radius: 50%;
                        background-color: #2ff0b0;
                        position: relative;
                        top: 200px;
                        animation-name: circle;
                        }   
                        @keyframes circle{
                     
                           0%{
                           left: 0px;
                           }
                     
                           50%{
                           border-radius: 0;
                           }
                     
                           100%{
                           left: 400px;
                           background-color: #34d0ff;
                           }
                        }

animation-duration

Отвечает за продолжительность анимации. Значения указываются в s и ms


animation-timing-function

Задаёт стиль анимации, то есть сценарий, по которому она будет осуществяться.

Некоторые значения:

  • ease - переход начинается медленно, разгоняется быстро и замедляется в конце.
  • ease-in - переход начинается медленно, а затем плавно ускоряется в конце.
  • ease-out - переход начинается быстро и плавно замедляется в конце.
  • ease-in-out - переход медленно начинается и медленно заканчивается.
  • linear - переход происходит равномерно на протяжении всего времени, без колебаний в скорости.

animation-iteration-count

Отвечает за количество повторов проигрывания анимации. Для бесконечного повторения используют animation-iteration-count: infinite


animation-direction

Определяет направление и тип проигрывания ключевых кадров.

Значения:

  • normal - ключевые кадры проиграются так, как они написаны.
  • reverse - кадры будут проигрываться в обратном порядке.
  • alternate - каждый чётный проход по кадрам будет проигрываться в обратном порядке.
  • alternate-reverse - каждый нечётный проход по кадрам будет проигрываться в обратном порядке.

animation-play-state

Позволяет запускать или останавливать анимацию

Значения:

  • running (по умолчанию) - анимация проигрывается.
  • paused - анимация на паузе.

animation-delay

Отвечает за задержку перед проигрыванием.


animation-fill-mode

Определяет какие значения анимируемых css свойств применятся к объекту после завершения анимации.

Значения:

  • none - никакие стили не применяются после анимации.
  • forwards - по окончании анимации элемент сохранит стили последнего ключевого кадра
  • backwards - по окончании анимации элемент сохранит стиль первого ключевого кадра
  • both - анимация будет вести себя так, как будто значения forwards и backwards заданы одновременно.
