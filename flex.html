Свойства контейнера

Свойства, которые применяются к контейнеру, действуют на все элементы в нём.

display

Значения:

  • flex - элементы выстраиваются в ряд. Контейнер остаётся блочным. При изменении ширины экрана флекс элементы не переносятся на другую строку.
  • inline-flex - контейнер похож на display: inline-block, контейнер тоже занимает столько места, сколько в нём контента. Если задать родителю text-align: center, то весь флекс контейнер центрируется по горизонтали.

justify-content

Устанавливает положение флекс элементов по основной линии (по той, по которой эти элементы расположены).

Значения:

  • flex-start — группа элементов начинается с левой границы контейнера. (значение по умолчанию)
  • flex-end — группа элементов заканчивается на правой границе
  • center - группа элементов располагается по центру
  • space-between (расстояние между) — первый элемент расположен в начале, последний — в конце. Все элементы между первым и последним разделяются пустым пространством. Не путайте с отступами. При изменении размеров экрана и уменьшении места, этого пространства может и не быть.
  • space-around (расстояние вокруг) — у каждого элемента расстояние слева и справа получается одинаковым. Расстояния не суммируются. Не путайте с отступами. При изменении размеров экрана и уменьшении места, этого пространства может и не быть.

align-items

Положение флекс элементов по неосновной линии.

Значения:

  • stretch — по высоте все флекс элементы подстраиваются под самый высокий. (значение по умолчанию)
  • flex-start — группа элементов начинается сверху
  • flex-end — группа элементов заканчивается снизу
  • center — группа элементов располагается по центру

flex-wrap

Перенос элементов на новую строчку при нехватке места.

Значения:

  • nowrap (по умолчанию) — не позволяет флекс элементам переноситься на новую строку при нехватке места
  • wrap — позволяет флекс элементам переноситься на новую строку при нехватке места
  • wrap-reverse — при нехватке места позволяет флекс элементам переноситься на новую строку, но в обратном порядке

flex

  • flex-basis - базовый размер элемента.auto — значение по умолчанию,
    где размер элемента равен размеру его контента. Значение можно задать в пикселях или в процентах.
  • flex-grow - возможность увеличиваться в размере относительно flex-basis
    Значения:
    0 — запрещено становиться больше flex-basis
    1 — разрешено становиться больше flex-basis
  • flex-shrink - возможность уменьшаться в размере относительно flex-basis
    Значения:
    0 — запрещено становиться меньше flex-basis
    1 — разрешено становиться меньше flex-basis

Сокращенная запись:

flex: flex-grow flex-shrink flex-basis
(flex: 1 1 auto приенить к main + flex-direction: column у body прижмёт футер к низу)

flex-direction

Задаёт основную ось внутри флекс контейнера

Значения:

  • row (по умолчанию) - выстраивает флекс элементы в ряд
  • row-reverse - выстраивает флекс элементы в ряд, но стартуют они с конца и в обратном порядке
  • column - выстраивает флекс элементы в колонку. Теперь основной линией (по той, по которой эти элементы расположены) стала вертикаль, а значит justufy-content располагает элементы по вертикали, а align-items - по горизонтали.
  • column-reverse - выстраивает флекс элементы в колонку, но стартуют они с конца и в обратном порядке
