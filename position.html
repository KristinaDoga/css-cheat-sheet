Позиционирование

Основное свойство позиционирования — position. Устанавливает положение элемента относительно других элементов или относительно окна браузера.

position

Значения:

  • static — значение по умолчанию. Так отображаются все элементы, если указано position: static или свойство position не указано вообще.
  • relative — позиционирование относительно изначального положения в коде.
  • fixed — позиционирование относительно окна браузера. Обязательно использование top, left, right или bottom
  • absolute — элемент удаляется из кода, место под него не резервируется. Сам элемент позиционируется относительно своего ближайшего позиционированного родителя, если такового нет, то относительно окна браузера.
    Обязательнo использовать вместе с top, left, right или bottom + position: relative у родительского элемента.
  • sticky — элемент позиционируется как static до опредёленной позиции, а затем смещается относительно его ближайшего родителя

  • Дополнительные свойства top, left, right, bottom нужны, чтобы указать положение элемента.
  • Дополнительное свойство z-index oтвечает за расположение элементов по оси z. То есть за перекрытие элементами друг друга, если они находятся на одном и том же месте страницы.
    Принимает целые значения от нуля, то есть 0,1,2...
    По умолчанию у всех элементов z-index: 0.
    z-index игнорируется для position: static
